# Project Overlay Template

Copy the files in this folder into a real project when you want repo-local instructions on top of the shared `ClaudeCodePilot` pack.

Suggested usage:

- copy `CLAUDE.md` if you also use Claude-family tools
- copy `AGENTS.md` if you want repo-local AI conventions in a portable format
- copy `.github/copilot-instructions.md` for always-on Copilot workspace guidance
- copy and adapt the sample `.github/instructions/*.instructions.md` files for path-specific behavior
- copy the sample `.github/prompts/project-audit.prompt.md` if you want a repo-specific audit entry point that assumes this pack is installed

Replace placeholder text with real commands, architecture notes, and team conventions.
